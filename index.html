Queries

This section present a list of relevant queries against WRITE KB, which can be run in this page or in the SPARQL endpoint section. The list of queries defines WRITE KB potentialities in storing and retriving data. The queries presented in this section are mostly focues over calligraphic components of artworks and their artistic and linguistic analysis.

We recently submitted WRITE KB at ADHO2022 conference. For this reason credits and project affiliation has been temporary hidden.
Disclamer: WRITE KB is still under development, WRITE domain experts are still working on the data entry activity. For this reason data will be updated.
