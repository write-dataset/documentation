WRITE data model


Chinese tradition, culture and art represent a certain challenge when it comes to the application of Digital Humanities' practices. Even though DH literature on Linked Open Data in the cultural heritage is extremly rich, we can still observe the lack of studies and publications that tackle data models used for representation of Chinese calligraphy, both artisticly and linguistically.

Thus, WRITE data model contains both reused and made ad hoc classes and properties, in order to fully represent the complex phenomenon of Chinese contemporary calligraphy.

WRITE Knowledge Base is populated with elements from 14 classes (owl:Class), 10 of which have been reused from Wikidata, while other 4 were created on purpose. This documentation illustrates these 14 classes along with their respective properties (owl:ObjectProperty and owl:DataTypeProperty).

Each of the four WRITE Collections, namely Visual Arts, Graffiti Art, Decorative & Applied Arts and Performing Arts, are represented with a dedicated class. It was important to preserve the diversity between them, in order to emphasise their individual peculiar elements.

For example, when describing artworks belonging to the class Graffiti, it is important to record particularities related specifically to this type of artwork, such as the graffiti style or the presence of a chinese visual element. On the other hand, WRITE data model seeks to express the complexity of Performing arts, given that the realisation of a performance can employ numerous external artworks, as well as different actors in several roles. Furthermore, Decorative & Applied arts can offer insight into the commercial and economical aspects of these artworks, while Visual artworks in this data set demonstrate both interconnection with other artworks and modern tendencies related to digital arts.

Data entry process is being done using Omeka S platform. Here, the term Resource Template is always referred to as a synonym for owl:Class. These resource templates are fully customisable and can be enriched with various metadata fields. Once filled and saved, a resource template becomes an item of the dataset, as an individual belonging to a class. Visit Omeka S website for more information.

In this documentation, each Class is presented with a brief introduction and a corresponding image that graphically summarises all the possible connections with other Classes/Resource Templates (owl:Class, via owl:ObjectProperty), as well as the metadata fields associated with the class in question.

All the images can be zoomed in for clearer exploration.

List of Classes with respective metadata


Visual Artwork resource template (owl:Class) represents Visual arts items. The individuals of this class constitute one of the four collections of the WRITE KB.

Decorative and Applied Arts resource template (owl:Class) represents Decorative and Applied arts items. The individuals of this class constitute one of the four collections of the WRITE KB.

Graffiti resource template (owl:Class) represents Graffiti art items. The individuals of this class constitute one of the four collections of the WRITE KB.

Performing Arts resource template (owl:Class) represents Performing arts items. The individuals of this class constitute one of the four collections of the WRITE KB.

Series of Creative Works resource template (owl:Class) represents an explicit connection between artworks of the four WRITE collections with the work series they may belong to.

Calli-Writing Unit (owl:Class) is a peculiar entity that aims to represent the calligraphic characteristics of the artworks. These units are to be considered integral parts of an artwork that contain some kind of calligraphy. However, since the contemporary calligraphy has been modifying the traditional one through these modern productions, the research team could not denominate nor cathegorise a priori this unit as a "calligraphic unit". Proposed metadata fields aim to dispose the Knowledge Base for an analysis on whether and to what measure traditional and contemporary Chinese calligraphies are similar or divergent.

Literary Work resource template (owl:Class) represents any literary production connected to a calli-writing unit, since the content of a calli-writing unit could have been based on a certain literary piece.

Traditional Artwork is a resource template (owl:Class) that represents artworks made before 1980 that are somehow related to the contemporary artworks of the 4 WRITE collections.

Person resource template (owl:Class) represents the complex network of agents intended as singles that operate in the context of this project: calligraphers, artists, performers, dancers, musicians, authors, curators, organisers and so on.

Organisation resource template (owl:Class) represents the network of agents intended as institutions that are involved in this project: universities, departments, research centres, museums, art galleries, brand companies etc.

Artistic collective resource template (owl:Class) represents the network of agents intended as groups that operate in the context of this project: crews, dance companies, design studios, architectural firms, tattoo studios, etc.

Exhibition resource template (owl:Class) represents the exhibition(s) in which a certain artwork from the four WRITE collections was displayed.

Publication is a resource template (owl:Class) that represents publications and archival documents that can be in any way related to the artworks of the 4 WRITE collections, the artists, creators, calligraphers and other actors of the KB, or exhibitions.

Life Event resource template (owl:Class) is still under development. It ought to describe more precisely the artistic and academic journey of the artists of the WRITE knowledge base, in order to analyse the socio-political aspect of the Chinese contemporary calligraphy phenomenon. An interesting aspect to investigate in the future could be to look at various influences across contintents in the life and work of these artists, such as European and American influeces.
